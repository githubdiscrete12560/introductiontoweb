Strong Induction Proof

For any tree T: |E| = |V| - 1

An Interactive Mathematical Proof

Theorem Statement

Theorem: For any tree T with n vertices, T has exactly n-1 edges.

Method: We'll prove this using strong induction on n

Why strong induction? Because when we decompose a tree into subtrees, we get components of various sizes - not just (n-1). We need our hypothesis to work for all smaller sizes.

Base Case (n = 1)

v
A single-vertex tree has 0 edges
|E| = 0 = 1 - 1 ✓

The base case is trivially satisfied.

Strong Inductive Hypothesis

Assume for some k ≥ 1 that every tree with m vertices satisfies
|E| = m - 1 for all 1 ≤ m ≤ k

Key point: This is strong induction because we assume the theorem holds for all tree sizes from 1 up to k, not just size k.

This gives us the flexibility to apply our hypothesis to subtrees of any size ≤ k.

Inductive Step Setup

Goal: Prove the theorem for trees with k+1 vertices

Let T be any tree with |V(T)| = k + 1
  • Root T at an arbitrary vertex r
  • Let the subtrees hanging from r be T₁, T₂, ..., Tₜ
  • Write nᵢ = |V(Tᵢ)| for each subtree i

This decomposition is the key to applying strong induction!

Key Observations

Each nᵢ ≥ 1
Σ(i=1 to t) nᵢ = k

Why? Because T has k+1 vertices total, and one of them is the root r.

Crucial insight: Each subtree Tᵢ is a tree with nᵢ ≤ k vertices, so we can apply our inductive hypothesis to each one!

Applying Inductive Hypothesis

Since each Tᵢ has nᵢ ≤ k vertices,
by the inductive hypothesis:

|E(Tᵢ)| = nᵢ - 1 for each i

This is where strong induction is essential! We're applying our hypothesis to multiple subtrees of different sizes simultaneously.

Counting Total Edges

The edges of T consist of:

  • All edges inside the subtrees T₁, T₂, ..., Tₜ
  • Plus the t edges connecting root r to its children
|E(T)| = Σ|E(Tᵢ)| + t

Now we substitute what we know from the inductive hypothesis...

The Calculation

|E(T)| = Σ|E(Tᵢ)| + t
= Σ(nᵢ - 1) + t [by inductive hypothesis]
= (Σnᵢ) - t + t
= Σnᵢ
= k [from our observation]
Since |V(T)| = k + 1, we have:
|E(T)| = k = (k + 1) - 1 = |V(T)| - 1 ✓

Conclusion

✓ By strong induction, the statement holds for all n ≥ 1

Why was strong induction the right choice?

  • We decomposed the tree into multiple subtrees of different sizes
  • We needed our hypothesis to apply to all these different sizes
  • Strong induction gave us that flexibility

The proof is complete! 🎉
