üí° What are Eigenvalues & Eigenvectors?

Eigenvector is a special vector that, when multiplied by matrix A, results in a vector with the same direction but possibly different magnitude

Eigenvalue is the number that tells us how much the vector is scaled

üîç What is PCA?

PCA is a technique that uses Eigenvalues and Eigenvectors to:

  • Reduce data dimensionality
  • Find directions of maximum data variance
  • Remove unimportant information
  • Create Principal Components
