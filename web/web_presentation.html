Introduction to the Internet

What is the Internet?

  • Global network of interconnected computers
  • Uses standardized communication protocols (TCP/IP)
  • Enables worldwide information sharing and communication
  • Infrastructure supporting various services and applications

Key Components

Hardware

Servers, routers, cables, data centers

Software

Operating systems, web browsers, applications

Protocols

HTTP/HTTPS, TCP/IP, DNS, FTP

Standards

HTML, CSS, JavaScript, XML

Additional Technical Details

Network Infrastructure

Internet backbone consists of high-capacity fiber optic cables, submarine cables connecting continents, and satellite links for remote areas.

Protocol Stack

The Internet operates on a layered model: Physical layer, Data link layer, Network layer (IP), Transport layer (TCP/UDP), and Application layer (HTTP, HTTPS, FTP).

Domain Name System

DNS translates human-readable domain names into IP addresses, enabling users to access websites using memorable names instead of numeric addresses.

Internet Service Providers

ISPs provide internet connectivity to end users, ranging from local providers to global telecommunications companies.

Historical Development

  • 1969: ARPANET, the precursor to the Internet, was created
  • 1983: TCP/IP protocol became standard
  • 1989: Tim Berners-Lee invented the World Wide Web
  • 1993: First web browser with graphical interface (Mosaic)
  • 1995: Internet became commercialized and public

บทนำสู่อินเทอร์เน็ต

อินเทอร์เน็ตคืออะไร?

  • เครือข่ายคอมพิวเตอร์ที่เชื่อมต่อกันทั่วโลก
  • ใช้โปรโตคอลการสื่อสารมาตรฐาน (TCP/IP)
  • ช่วยให้สามารถแบ่งปันข้อมูลและสื่อสารทั่วโลก
  • โครงสร้างพื้นฐานที่รองรับบริการและแอปพลิเคชันต่างๆ

องค์ประกอบสำคัญ

ฮาร์ดแวร์

เซิร์ฟเวอร์, เราเตอร์, สายเคเบิล, ศูนย์ข้อมูล

ซอฟต์แวร์

ระบบปฏิบัติการ, เว็บเบราว์เซอร์, แอปพลิเคชัน

โปรโตคอล

HTTP/HTTPS, TCP/IP, DNS, FTP

มาตรฐาน

รายละเอียดทางเทคนิคเพิ่มเติม

โครงสร้างพื้นฐานเครือข่าย

แกนหลักของอินเทอร์เน็ตประกอบด้วยสายเคเบิลใยแก้วนำแสงความจุสูง สายเคเบิลใต้น้ำเชื่อมต่อทวีป และการเชื่อมต่อผ่านดาวเทียมสำหรับพื้นที่ห่างไกล

Protocol Stack

อินเทอร์เน็ตทำงานบนโมเดลแบบชั้น: ชั้นกายภาพ ชั้นการเชื่อมโยงข้อมูล ชั้นเครือข่าย (IP) ชั้นการขนส่ง (TCP/UDP) และชั้นแอปพลิเคชัน (HTTP, HTTPS, FTP)

ระบบชื่อโดเมน

DNS แปลงชื่อโดเมนที่มนุษย์อ่านได้เป็นที่อยู่ IP ทำให้ผู้ใช้สามารถเข้าถึงเว็บไซต์โดยใช้ชื่อที่จำง่ายแทนที่จะเป็นตัวเลข

ผู้ให้บริการอินเทอร์เน็ต

ISP ให้บริการการเชื่อมต่ออินเทอร์เน็ตแก่ผู้ใช้ปัจจุบัน ตั้งแต่ผู้ให้บริการท้องถิ่นไปจนถึงบริษัทโทรคมนาคมระดับโลก

การพัฒนาทางประวัติศาสตร์

  • 1969: ARPANET บรรพบุรุษของอินเทอร์เน็ตถูกสร้างขึ้น
  • 1983: โปรโตคอล TCP/IP กลายเป็นมาตรฐาน
  • 1989: Tim Berners-Lee ประดิษฐ์ World Wide Web
  • 1993: เว็บเบราว์เซอร์แรกที่มีอินเทอร์เฟซแบบกราฟิก (Mosaic)
  • 1995: อินเทอร์เน็ตกลายเป็นเชิงพาณิชย์และสาธารณะ

อินเทอร์เน็ต vs. เวิลด์ไวด์เว็บ

อินเทอร์เน็ต: โครงสร้างพื้นฐานของเครือข่าย

เว็บ: บริการที่ทำงานบนอินเทอร์เน็ตโดยใช้โปรโตคอล HTTP

Web 1.0 - The Static Web

(1990s-2000s)

Characteristics

  • Read-only web - Static content consumption
  • Simple HTML pages with basic formatting
  • Limited interactivity and user engagement
  • Centralized content creation by webmasters

Key Features

  • Static web pages with text and images
  • Basic HTML and CSS styling
  • Simple navigation through hyperlinks
  • Email and basic forms for communication
  • Directory-based websites (Yahoo!, DMOZ)

Examples

Early corporate websites

Online brochures and catalogs

Simple news websites

Basic e-commerce sites

Limitations

  • No user-generated content
  • Limited multimedia support
  • Slow dial-up connections
  • Minimal personalization

Web 1.0 - เว็บแบบคงที่

(ทศวรรษ 1990-2000)

ลักษณะเด่น

  • เว็บแบบอ่านอย่างเดียว - การบริโภคเนื้อหาแบบคงที่
  • หน้าเว็บ HTML แบบง่ายๆ ที่มีการจัดรูปแบบพื้นฐาน
  • การโต้ตอบและการมีส่วนร่วมของผู้ใช้อย่างจำกัด
  • การสร้างเนื้อหาแบบรวมศูนย์โดยเว็บมาสเตอร์

คุณสมบัติสำคัญ

  • หน้าเว็บแบบคงที่ที่มีข้อความและรูปภาพ
  • การจัดแต่ง HTML และ CSS พื้นฐาน
  • การนำทางแบบง่ายผ่านไฮเปอร์ลิงก์
  • อีเมลและฟอร์มพื้นฐานสำหรับการสื่อสาร
  • เว็บไซต์แบบไดเรคทอรี (Yahoo!, DMOZ)

ตัวอย่าง

เว็บไซต์บริษัทยุคแรก

โบรชัวร์และแคตตาล็อกออนไลน์

เว็บไซต์ข่าวแบบง่าย

เว็บไซต์อีคอมเมิร์ซพื้นฐาน

ข้อจำกัด

  • ไม่มีเนื้อหาที่สร้างโดยผู้ใช้
  • การรองรับมัลติมีเดียอย่างจำกัด
  • การเชื่อมต่อแบบ dial-up ที่ช้า
  • การปรับแต่งส่วนบุคคลขั้นต่ำ
