🌐Internet

  • What: Global network infrastructure
  • When: Started in 1969 (ARPANET)
  • Protocols: TCP/IP (foundation)
  • Purpose: Connect devices worldwide
  • Scope: All digital communications

🕸️Web

  • What: Information system service
  • When: Created in 1989-1991
  • Protocols: HTTP/HTTPS (specific)
  • Purpose: Share information via browsers
  • Scope: Websites and web pages

💡 Key Insight

The Web is just one service that runs on top of the Internet infrastructure. Many other services also use the Internet, like email, file transfers, and online gaming!

🌐 The Internet: Global Infrastructure

🏗️Internet = Foundation

The Internet is the underlying network infrastructure that connects billions of devices worldwide using standardized protocols.

🖥️

Physical Layer

Cables, routers, servers, data centers

📡

Network Protocols

TCP/IP, DNS, routing protocols

🔗

Global Connectivity

Interconnected networks worldwide

⚡

Data Transmission

Packet routing and delivery

🔧 What the Internet Provides:

  • Physical connectivity: Cables, wireless signals, satellites
  • Addressing system: IP addresses to identify devices
  • Routing mechanism: Moving data from source to destination
  • Protocol standards: Rules for how devices communicate
  • Reliability: Redundant paths and error correction

📱 Real Example:

When you send a WhatsApp message, you're using the Internet infrastructure (TCP/IP, routers, cables) but not the Web (no HTTP, no browser, no websites involved).

🕸️ The Web: Information System

📄Web = Service

The World Wide Web is an information system that uses the Internet infrastructure to share documents and media through web browsers.

🌐

HTTP/HTTPS

Protocols for web communication

📄

Web Pages

HTML documents with content

🔗

Hyperlinks

Connections between pages

🌍

Web Browsers

Applications to access the Web

🔧 What the Web Provides:

  • Web pages: HTML documents with text, images, videos
  • Hyperlinks: Clickable connections between pages
  • URLs: Web addresses (like www.example.com)
  • Browsers: Applications to view web content
  • Websites: Collections of related web pages

🌐 Real Example:

When you visit www.google.com in your browser, you're using both the Internet (for connectivity) and the Web (HTTP protocol, HTML pages, hyperlinks).

👨‍💻 Created by Tim Berners-Lee

Web History:

1989: Tim Berners-Lee proposes the World Wide Web
1990: First web browser and web server created
1991: First website goes live at CERN
1993: Web becomes freely available to everyone

📅 Historical Timeline

1969

ARPANET Born

The predecessor to the Internet connects 4 computers. This is the beginning of the Internet infrastructure.

1983

TCP/IP Adopted

The Internet Protocol Suite becomes the standard, creating the modern Internet foundation.

1989

Web Proposal

Tim Berners-Lee proposes the World Wide Web - a new service to run on the existing Internet.

1991

First Website

The first website goes live, marking the birth of the Web as we know it.

1993

Web Goes Public

CERN announces that the Web will be free for everyone to use.

⏰ Key Insight:

The Internet existed for 20 years before the Web was invented! During this time, people used the Internet for email, file transfers, and remote access - but no websites existed yet.

🌍 Real-World Examples

🌐 Using the Web (Internet + HTTP)

Action: Browsing www.wikipedia.org

Internet part: TCP/IP protocols, DNS lookup, network routing

Web part: HTTP request, HTML response, browser rendering

📧 Using Email (Internet, NOT Web)

Action: Sending email via Gmail app

Internet part: TCP/IP protocols, network connectivity

NOT Web: Uses SMTP/IMAP, not HTTP; no browser needed

🎮 Online Gaming (Internet, NOT Web)

Action: Playing Fortnite online

Internet part: Network connectivity, data transmission

NOT Web: Custom game protocols, no websites involved

📱 Mobile App (Internet, Sometimes Web)

Action: Using a banking app

Internet part: Network connectivity for data

Maybe Web: If app uses HTTP APIs (hybrid approach)
