Introduction to the Internet

What is the Internet?

  • Global network of interconnected computers
  • Uses standardized communication protocols (TCP/IP)
  • Enables worldwide information sharing and communication
  • Infrastructure supporting various services and applications

Key Components

  • Hardware: Servers, routers, cables, data centers
  • Software: Operating systems, web browsers, applications
  • Protocols: HTTP/HTTPS, TCP/IP, DNS, FTP
  • Standards: HTML, CSS, JavaScript, XML

Internet vs. World Wide Web

  • Internet: The underlying network infrastructure
  • Web: Service that runs on top of the Internet using HTTP protocol

Web 1.0 - The Static Web (1990s-2000s)

Characteristics

  • Read-only web - Static content consumption
  • Simple HTML pages with basic formatting
  • Limited interactivity and user engagement
  • Centralized content creation by webmasters

Key Features

  • Static web pages with text and images
  • Basic HTML and CSS styling
  • Simple navigation through hyperlinks
  • Email and basic forms for communication
  • Directory-based websites (Yahoo!, DMOZ)

Examples

  • Early corporate websites
  • Online brochures and catalogs
  • Simple news websites
  • Basic e-commerce sites

Limitations

  • No user-generated content
  • Limited multimedia support
  • Slow dial-up connections
  • Minimal personalization

Web 2.0 - The Social Web (2000s-2010s)

Characteristics

  • Read-write web - User participation and interaction
  • Dynamic content and social networking
  • Collaborative platforms and user-generated content
  • Rich user interfaces and multimedia integration

Key Technologies

  • AJAX (Asynchronous JavaScript and XML)
  • Rich Internet Applications (RIAs)
  • RSS feeds and syndication
  • APIs and web services
  • Content Management Systems (CMS)

Examples

  • Social media platforms (Facebook, Twitter, YouTube)
  • Blogs and wikis (WordPress, Wikipedia)
  • Photo and video sharing (Flickr, Instagram)
  • Collaborative tools (Google Docs, Dropbox)

Impact

  • Rise of social networking
  • User-generated content explosion
  • Emergence of online communities
  • New business models (advertising, freemium)
