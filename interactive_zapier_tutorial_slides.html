Step 3

Set Up Gmail Trigger

1

Choose Gmail

Select Gmail as your trigger app

2

Select Event

Choose "New Email" trigger

3

Connect Account

Authenticate your Gmail

4

Configure Settings

Set up email filters (optional)

🔧 Configuration Options:

Label: Leave blank for all emails

Search String: Use filters like "is:unread" or "from:specific@email.com"

For beginners: Leave both fields blank to capture all new emails

Step 5

Map Your Data Fields

📅

Date Column

Gmail "Date" → Sheet Column A

👤

Sender Column

Gmail "From Email" → Sheet Column B

📝

Subject Column

Gmail "Subject" → Sheet Column C

Data Mapping Example: Date: {{125798543__date}} Sender: {{125798543__from__email}} Subject: {{125798543__subject}}

🎯 Mapping Tips:

• Click in each field and select the corresponding Gmail data

• Use "From Email" for email address or "From Name" for display name

• Each field should show a reference like {{data_field}}

Common Issues & Solutions

❌

Zap Not Triggering

Gmail connection issues

📭

No Data in Sheet

🔄

Wrong Data Format

Field mapping issues

⚠️

Task Limit Reached

Free tier limitations

🔧 Quick Fixes:

Connection Issues: Reconnect your Gmail/Google accounts

Permission Denied: Check Google Sheet sharing settings

No New Emails: Zap only processes emails received AFTER activation

Wrong Sheet: Verify you selected the correct spreadsheet and worksheet
