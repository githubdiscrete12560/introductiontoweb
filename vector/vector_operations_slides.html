Matrix as Vectors

Two Perspectives

Think of matrix A as collection of column vectors OR row vectors

Column view: A = [a‚ÇÅ a‚ÇÇ ... a‚Çô], then Ax = x‚ÇÅa‚ÇÅ + x‚ÇÇa‚ÇÇ + ... + x‚Çôa‚Çô

Row view: Each entry of Ax is dot product of row with x

Example 1: Column View

Calculate Ax using column view:

[3 4] [-1]

Column 1: [1, 3], Column 2: [2, 4]

Ax = 5[1, 3] + (-1)[2, 4] = [5, 15] + [-2, -4] = [3, 11]

Linear combination of columns!

Example 2: Row View

Same calculation using row view:

A = [1 2] x = [5]
[3 4] [-1]

Row 1: [1, 2] ¬∑ [5, -1] = 1√ó5 + 2√ó(-1) = 5 - 2 = 3

Row 2: [3, 4] ¬∑ [5, -1] = 3√ó5 + 4√ó(-1) = 15 - 4 = 11

Result: Ax = [3, 11] - same answer!

Example 3: Larger Matrix

Column view calculation:

[0 1 3] [4]
[-2]

Determinants in 2D & 3D

üéØ Geometric Meaning

Determinant measures how much a transformation scales areas (2D) or volumes (3D)

Sign indicates orientation: positive preserves, negative flips

2D: det([a b; c d]) = ad - bc

3D: det(A) = a‚ÇÅ‚ÇÅ(a‚ÇÇ‚ÇÇa‚ÇÉ‚ÇÉ - a‚ÇÇ‚ÇÉa‚ÇÉ‚ÇÇ) - a‚ÇÅ‚ÇÇ(a‚ÇÇ‚ÇÅa‚ÇÉ‚ÇÉ - a‚ÇÇ‚ÇÉa‚ÇÉ‚ÇÅ) + a‚ÇÅ‚ÇÉ(a‚ÇÇ‚ÇÅa‚ÇÉ‚ÇÇ - a‚ÇÇ‚ÇÇa‚ÇÉ‚ÇÅ)

Key Applications:

  • Area/volume scaling in transformations
  • Checking matrix invertibility (det ‚â† 0)
  • Orientation testing (clockwise vs counterclockwise)
  • Cross product magnitude in 3D

Calculate determinant using cofactor expansion:

[0 4 1]
[1 2 0]

Example 4: Cross Product and Determinant Connection

Show that ||a√ób|| equals determinant of 2√ó2 matrix

For a = (3,4,0) and b = (1,2,0) in xy-plane
